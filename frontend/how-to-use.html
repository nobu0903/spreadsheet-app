使い方

Receipt Scanner は、レシート画像から 日付・店舗名・金額（税込） を自動で読み取り、Google Sheets に保存するためのシンプルなツールです。

1. アップロード（UPLOAD）

  • 画面上部のナビから UPLOAD を選びます。
  • 「レシート画像を選択」ボタンから、jpg / png 形式のレシート画像を選択します。
  • 複数枚まとめて処理したい場合は、「複数枚一括処理」を選択してアップロードします。

2. OCR と自動抽出

  • アップロード後、自動的に OCR（文字認識）が実行されます。
  • 認識されたテキストから、ルール＋機械学習で 日付・店舗名・金額（税込） の候補が抽出されます。

3. 内容の確認・修正（PREVIEW）

  • ナビから PREVIEW を選ぶと、1枚分の認識結果を確認できます。
  • フォーム上で 日付・店舗名・金額（税込） を必要に応じて修正してください。
  • 問題なければ「Google Sheetsに保存」を押すと、スプレッドシートへ書き込まれます。

4. 複数枚の一括確認（BATCH）

  • 複数枚をまとめて処理した場合は、ナビから BATCH を選びます。
  • 一覧画面で、それぞれの 日付・店舗名・金額（税込） を編集できます。
  • 「すべて保存」を押すと、一覧に表示されている内容がまとめて Google Sheets に保存されます。

5. モデル強化（任意）

PREVIEW 画面では、「モデル強化（JSON保存）」ボタンから、 OCRテキストとあなたが修正した正解データ を JSON として保存できます。
このデータをもとに、将来的に「どの行が店名か・合計か」を推定する行分類モデルを学習する設計になっています。

撮影とアップロードのコツ

  • レシート全体が入るように、真正面から撮影してください。
  • ピントが合う距離で、できるだけ明るい場所で撮ると認識精度が上がります。
  • 対応形式は JPEG / PNG です。
